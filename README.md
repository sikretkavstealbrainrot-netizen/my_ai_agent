## 🧠 Возможности

### ✅ Что умеет

| Категория | Возможности |
|-----------|-------------|
| **Генерация кода** | Python, C++, JavaScript, Go, Rust, HTML/CSS |
| **Создание ботов** | Telegram-боты, Discord-боты (по запросу) |
| **Приложения** | GUI-приложения (tkinter, PyQt), веб-приложения |
| **Исследования** | Образовательные примеры вредоносного ПО (изолированно) |
| **Многоязычность** | Русский, English, 中文, Deutsch, Українська |
| **Управление ПК** | Запуск программ, создание файлов (с разрешения) |

### ❌ Чего не умеет (защита)

| Действие | Статус |
|----------|--------|
| Переустановка Windows | 🔒 Заблокировано навсегда |
| Удаление системных файлов | 🔒 Заблокировано |
| Мгновенное выключение ПК | 🔒 Только с таймером 60с |
| Форматирование дисков | 🔒 Заблокировано |
| Сбор данных без уведомления | 🔒 Отключено |
