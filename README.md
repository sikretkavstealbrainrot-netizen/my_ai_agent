## 🧠 Возможности

### ✅ Что умеет

| Категория | Подробности |
|-----------|-------------|
| **💻 Генерация кода** | Python, C++, JavaScript, Go, Rust, HTML/CSS, Bash |
| **🤖 Создание ботов** | Telegram-боты (полный код), Discord-боты |
| **🖼️ GUI приложения** | Tkinter, PyQt, веб-интерфейсы |
| **🔬 Исследования** | Образовательные примеры (вирусы, эксплойты) с предупреждениями |
| **🌍 Многоязычность** | Русский, English, 中文, Deutsch, Українська, Español |
| **🖱️ Управление ПК** | Запуск программ, создание/чтение файлов (с разрешения) |
| **📁 Работа с файлами** | Чтение, запись, сохранение кода в `generated/` |

### ❌ Чего НЕ умеет (безопасность)

| Действие | Статус |
|----------|--------|
| Переустановка Windows | 🔒 **Заблокировано навсегда** |
| Удаление системных файлов | 🔒 **Заблокировано** |
| Мгновенное выключение ПК | 🔒 Только с таймером 60с |
| Форматирование дисков | 🔒 **Заблокировано** |
| Скрытый сбор данных | 🔒 **Отключено** |
