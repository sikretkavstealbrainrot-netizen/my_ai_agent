# 🤖 My AI Agent — Локальный ИИ-ассистент с расширенными возможностями

**Бюджет: $0** | **Работает оффлайн** | **Полный контроль кода**

## 🧠 Возможности
- ✅ Генерация кода (Python, C++, JavaScript, Go, Rust)
- ✅ Создание Telegram-ботов
- ✅ Написание приложений и скриптов (в т.ч. исследовательских)
- ✅ Управление ПК (с разрешения пользователя)
- ✅ Многоязычность: русский, English, 中文, Deutsch, Українська

## ⚠️ Конституционные ограничения (нельзя даже с разрешения)
- ❌ Переустановка Windows
- ❌ Удаление системных файлов (C:\Windows, /System, /boot)
- ❌ Мгновенное выключение ПК (только с таймером 60с)
